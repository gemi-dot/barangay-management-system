# SOP Sheet - BHW Office
## Barangay Abgao IMS

## Purpose
Standard workflow for BHW health-related operations in Barangay IMS.

## Daily Start (AM)
1. Open browser and access system URL.
2. Login using BHW account only.
3. Check pending health updates and follow-up list.

## Core Duties
1. Health Record Updates
- Update health monitoring entries accurately.
- Maintain current status for senior citizens and pregnancy records.
- Record checkup dates and relevant conditions.

2. Report Maintenance
- Update BHW-related reports on schedule.
- Validate totals and key indicators before submission.
- Coordinate missing resident data with Secretary.

3. Health Coordination
- Flag urgent/high-risk cases for immediate action.
- Endorse concerns to Secretary and Captain as required.

## Quality Controls
1. Ensure resident identity matches the correct record.
2. Use current date and accurate medical notes.
3. Avoid duplicate health entries.
4. Confirm report fields are complete before finalizing.

## End-of-Day (PM)
1. Review newly added/edited health records.
2. Confirm report updates are saved.
3. Submit summary of pending follow-ups.

## Escalation Rules
- Data mismatch in resident identity: Secretary.
- Critical health concerns requiring action: Captain and health team.
- System access/problem: system administrator.

## Do Not
- Edit non-health resident master fields without coordination.
- Share account credentials.
- Encode unverified health information.
