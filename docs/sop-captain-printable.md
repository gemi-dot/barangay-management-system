# SOP Sheet - Barangay Captain Office
## Barangay Abgao IMS

## Purpose
Standard workflow for review, oversight, and decision support using Barangay IMS.

## Daily Start (AM)
1. Open browser and access system URL.
2. Login using Captain account only.
3. Review dashboard highlights and priority items.

## Core Duties
1. Dashboard and Report Review
- Check resident, household, and health summary indicators.
- Review trends and exceptions requiring policy action.
- Validate report outputs for meetings and planning.

2. Oversight and Approvals
- Review critical requests and endorsed cases.
- Provide direction for service prioritization.
- Ensure office compliance with data handling rules.

3. Coordination
- Align actions with Secretary and BHW updates.
- Confirm unresolved issues have assigned owners and due dates.

## Quality Controls
1. Compare key indicators across reports for consistency.
2. Monitor turnaround time for document requests.
3. Track high-risk cases and status of interventions.

## End-of-Day (PM)
1. Confirm priority actions are documented.
2. Review pending escalations from Secretary/BHW.
3. Set next-day directives for offices.

## Escalation Rules
- Data quality issue affecting official reports: Secretary.
- Public-health risk concerns: BHW and partner agencies.
- System reliability/access issue: system administrator.

## Do Not
- Share credentials.
- Approve decisions without supporting report data.
- Delay escalation of critical items.
