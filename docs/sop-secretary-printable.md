# SOP Sheet - Secretary Office
## Barangay Abgao IMS

## Purpose
Standard workflow for Secretary operations in Barangay IMS.

## Daily Start (AM)
1. Open browser and access system URL.
2. Login using Secretary account only.
3. Confirm dashboard loads and queue is visible.

## Core Duties
1. Resident Encoding
- Add new residents with complete identity and address details.
- Verify correct Purok selection before saving.
- Correct duplicates or inconsistent entries promptly.

2. Document Request Processing
- Open document requests queue.
- Review request details and completeness.
- Update status (pending, processing, ready for pickup, released, rejected).
- Record OR number/control number when applicable.

3. Certificate Printing
- Open certificate page from request record.
- Confirm resident details and purpose.
- Verify Captain name and office location values.
- Print and release according to office policy.

## Quality Controls
1. No blank mandatory fields.
2. Check spelling of names and addresses.
3. Confirm contact number format.
4. Validate request status before release.

## End-of-Day (PM)
1. Verify all processed requests are status-updated.
2. Run daily backup procedure (assigned workstation).
3. Confirm backup file was created.
4. Log unresolved issues for next day.

## Escalation Rules
- Health-data concerns: coordinate with BHW.
- Policy/approval concerns: coordinate with Captain.
- System issue: notify system administrator immediately.

## Do Not
- Share login credentials.
- Use another office account.
- Delete records without authorization.
